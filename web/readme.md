## Acesse a app pelo link

1. Clone o repositório
```
git clone https://github.com/BiaCarvalhoCavalieri/trainees-fifth-proof
```

2. Entre na pasta web
```
cd web
```

3. Install dependencies
```
npm install
```

4. Execute the front
```
npm start
```

5. Entre no página do checkout
```
/checkout
```
